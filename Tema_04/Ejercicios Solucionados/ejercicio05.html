Ejercicio 05 - unshift() y shift()

Enunciado: Dado el array cola, añade una persona al inicio con unshift() (desde el input) y luego elimina la primera con shift(). Muestra resultados.

Pulsa Ejecutar. Después podrás mostrar el código fuente con el botón.

(aquí aparecerá el resultado)

Código fuente del ejercicio

function $(sel) { return document.querySelector(sel); }
  function out(msg) { $("#output").textContent = String(msg); }


function run() {
  var cola = ["Ana", "Luis", "Carlos"];
  var p = $("#persona").value;
  p = String(p).trim();
  if (p.length === 0) {
    out("Escribe un nombre.");
    return;
  }

  cola.unshift(p);
  var atendida = cola.shift();

  out("Atendida (shift) = " + atendida + "\nCola final = " + cola.join(", "));
}

$("#run").addEventListener("click", run);
