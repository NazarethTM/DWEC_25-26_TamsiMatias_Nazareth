Ejercicio 04 - pop()

Enunciado: Dado el array tareas, elimina el último elemento con pop() y muestra el elemento eliminado y el array final.

Pulsa Ejecutar. Después podrás mostrar el código fuente con el botón.

Salida

(aquí aparecerá el resultado)

Código fuente del ejercicio

function $(sel) { return document.querySelector(sel); }
  function out(msg) { $("#output").textContent = String(msg); }


function run() {
  var tareas = ["estudiar", "comprar", "entrenar"];
  var eliminado = tareas.pop();
  out("eliminado = " + eliminado + "\ntareas = " + tareas.join(", "));
}

$("#run").addEventListener("click", run);
