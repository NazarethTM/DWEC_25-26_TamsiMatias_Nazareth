Ejercicio 09 - indexOf()

Enunciado: Comprueba si el nombre escrito en el input está en el array nombres usando indexOf(). Si está, muestra su posición; si no, indica que no existe.

Pulsa Ejecutar. Después podrás mostrar el código fuente con el botón.

(aquí aparecerá el resultado)

Código fuente del ejercicio

function $(sel) { return document.querySelector(sel); }
  function out(msg) { $("#output").textContent = String(msg); }


function run() {
  var nombres = ["Ana", "Luis", "Carlos", "Marta"];
  var q = $("#q").value;
  q = String(q).trim();
  if (q.length === 0) {
    out("Escribe un nombre a buscar.");
    return;
  }
  var pos = nombres.indexOf(q);
  if (pos === -1) {
    out("nombres = " + nombres.join(", ") + "\n\""+ q + "\" no existe en el array.");
  } else {
    out("nombres = " + nombres.join(", ") + "\n\""+ q + "\" está en la posición " + pos + ".");
  }
}

$("#run").addEventListener("click", run);
